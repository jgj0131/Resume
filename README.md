## 자기소개

```
 저는 새로운 지식에 대한 거리낌 없는 자세와 여러 현장 경험을 바탕으로 세상에 없던 것을 만들 수 있는 사람이라고 자신합니다. 아래와 같은 경험을 통해 함께 성장할 수 있다고 생각하며 많은 사람들이 편안하고 재미있는 세상에 살 수 있도록 하고 싶습니다.

 첫째, 고객의 소리를 직접 들어본 만큼 하나의 앱을 만들 때 사용자의 관점에서 생각하고 만들 수 있습니다. 개발자들은 자신이 만든 프로그램에 대해 피드백을 받을 기회가 많지 않습니다. 받더라도 앱의 리뷰 정도일 것이라 생각합니다. 하지만 저는 라운지랩에서 오프라인을 위한 앱을 만들어 본 경험이 있어 직접 사용자의 소리를 듣고 피드백을 반영하였습니다. 또한 개인 앱을 직접 사용자의 관점에서 기획하고 개발하였습니다. 현재까지도 사용자들의 리뷰를 리뷰를 꼼꼼하게 읽고, 사용자의 의견은 빠르게 반영하며 업데이트를 하였습니다.

 둘째, AI 개발자, 로봇 엔지니어 등 여러 분야의 사람들과 협업한 경험 덕분에 모든 분야의 사람들과 협업할 수 있다고 자신합니다. 보통 앱 개발자는 기획자, 디자이너, 서버 개발자와 함께 협업하는 일이 많습니다. 하지만 저는 LOUNGE'X, 무인상회와 같은 프로젝트를 진행하면서 바리스타, 로봇 엔지니어, AI 개발자 등 여러 분야의 사람들과 협업하였습니다. 새로운 분야에 대해 배우고 이해하려 노력했던 덕분에 항상 협업하는 동료의 입장에서 생각하고 배려하는 자세를 갖게 되었습니다.

 셋째, Swift, Java, Python 등 다양한 언어 경험은 구조를 설계함에 있어 좀 더 유연하게 생각하도록 도움을 줄 것입니다. 현장에서 직접 다양한 프로그램을 만들면서 언어별로 추구하는 방향이 다르다는 것을 알게 되었습니다. 이를 통해 각 프로그램에 대해 필요한 지식과 Framework도 익히게 되었습니다. Web과 App의 동작 방식의 차이, 그리고 같은 App이라 하더라도 android, iOS의 차이에 대해 직접 구현하고 테스트 하면서 시야를 넓히게 되는 계기가 되었습니다. 

 상기와 같이 다양하게 배우고 여러 사람들과 협업하며 기존에 없던 새로운 것들을 만들어낼 준비가 되어있다고 생각합니다. 좋은 구조, 수정이 쉬운 코드에 대해 함께 고민하는 동료이자 다른 직무에 대해 이해하고자 하는 자세를 가진 최고의 협력자, 그리고 사용하기 쉬우면서 삶에 큰 도움을 주는 앱을 만드는 개발자의 모습 꼭 보여드리겠습니다.
```
